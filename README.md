# Customer Segmentation using K-Means Clustering

## Project Overview
This project segments customers based on Age, Annual Income, and Spending Score using the K-Means clustering algorithm. The segmented data is visualized in a Power BI dashboard to support targeted marketing strategies.

## Objective
- Identify different customer groups.
- Help businesses personalize marketing campaigns.
- Improve customer engagement.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Power BI

## Dataset
Mall Customers Dataset

## Project Workflow
1. Data Cleaning
2. Feature Selection
3. Data Standardization
4. Elbow Method
5. K-Means Clustering
6. Dashboard Creation

## Dashboard Includes
- Total Customers
- Cluster Distribution
- Average Income by Cluster
- Average Spending Score by Cluster
- Average Age by Cluster
- Genre Distribution
- Interactive Filters

## Business Insights
- Customers were divided into five distinct segments.
- High-income, high-spending customers are ideal for premium marketing.
- Low-income, low-spending customers may benefit from discounts and loyalty programs.
- Different clusters require different marketing strategies.
